  • 『整個企劃背後的想法，其實可以追溯回我們認知中台灣「代工王國」定位。如今說起代工年代，多半帶些不知創新的貶義，但行人總編周易正提到：「我們覺得代工是台灣歷史與文化的一部分、是我們的一部分。我們應該面對它，並思考它在我們的文化中還能夠扮演什麼角色？可以如何繼續發展？」屬於我們的代工文化，養成一代台灣師傅高度的專注力及熟練的技術，更是台灣發展過程中不可或缺、未來也能以不同面貌留下來的文化。』。

  • 『這款遊戲是在1989年發行，憑藉著流暢的畫面、貼近真實的比賽畫面，使其成為許多人心目中的經典。如果你以為它只能用簡單的招數那就錯了，不同的按鍵組合可以做出許多不同的招式。例如發球除了一般的低手發球、高手發球，還能使用跳發、閃電球、太陽球等絢麗的方法；在攻擊上除有後排攻擊以外，甚至可以打時間差！（不過如果玩家操控的技術不佳，經常會出現揮空的窘境。）以下影片中比數11:6時的發球就是傳說中的閃電球！球達到最高點後畫面突然變亮加上球以閃電的ㄣ字型路徑下墜，許多玩家面對這樣的發球只能望球興嘆，堪稱是這遊戲的經典招數。』。

    • 關於排球的回憶：電玩「超級排球」

  • 『紅白機或 Gameboy 發達的年代，買上一台遊戲機是件奢侈的事。而無論香港或台灣，每天放學傍晚，總能看著男孩子們巍顫顫從口袋裡緩緩掏出，再小心翼翼地捧著，那是好不容易湊足的「規費」 ── 從父母配給得到，特別珍貴的餐費或零用錢。就算沒有吃飯，他們為的只是到錄影帶店或遊戲卡帶店，租上一片遊戲，徜徉在想像世界中，與頭目搏鬥，與角色戀愛、對話，更需要透過遊戲紀錄、技術與寶物，方能與朋友耀武揚威。』。

    • 香港遊戲二十年：《古龍群俠傳》《小朋友齊打交》以及我們的童年

  • 『Visual Studio Live Share讓開發團隊能夠在同樣的程式碼集（codebase）裡快速協作，不需要個別配置同樣的開發工具、設定或環境，也不必同步程式碼。當開發人員於Visual Studio 2017或Visual Studio Code撰寫程式時，只要按下「分享」鍵，它就會產生一組連結，將該連結傳遞給團隊成員，就能邀請他們加入該程式碼集進行協作。』之前會用 c9 這個平台也是看重線上協作這項功能，沒想到 VSCODE 也把它移植過來了，大推，隔壁棚的 Atom 也推出了打對台的 teletype，真的是太棒了，昔有 emacs 和 vim 大戰，今有 VSCODE 和 ATOM 爭奪編輯器之神的寶座。似乎都快忘了 IDE 的存在。
